# 独立研究闭环 模块

## 模块说明
- 独立目录：`frontend`、`backend`、`docker`、`docs`
- 独立后端：FastAPI + SQLite
- 独立前端：React + Vite + Zustand + React Query
- 独立职责：问题结构化分析、多线程并行搜索、迭代收敛、AI 报告提炼、网页端阅读

## 本地启动
### 后端
```bash
cd backend
python -m pip install -r requirements.txt
python -m uvicorn app.main:app --app-dir . --host 127.0.0.1 --port 8000
```

### 前端
```bash
cd frontend
npm install
npm run dev -- --host 127.0.0.1 --port 5174
```

## 测试
### 后端测试
```bash
cd backend
python -m pytest
```

### 前端测试
```bash
cd frontend
npm run check
npm run test
```

## 接口摘要
- `POST /api/v1/tasks`：创建研究任务
- `GET /api/v1/tasks`：查询任务列表
- `GET /api/v1/tasks/{taskId}`：查询任务详情
- `GET /api/v1/tasks/{taskId}/iterations`：查询迭代轨迹
- `GET /api/v1/tasks/{taskId}/report`：查询结构化报告
- `GET /api/v1/config/search-providers`：查询搜索源配置
- `GET /api/v1/config/ai`：查询 AI 配置
- `GET /api/v1/health`：健康检查

## 稳定性约束
- 长 任务走异步线程，不阻塞请求生命周期
- 搜索结果按维度做覆盖度评估，不达标继续迭代
- 报告生成失败时可回退到本地结构化总结
- 模块不依赖现有 `quant-system` 内部实现，仅通过 HTTP 边界交互
