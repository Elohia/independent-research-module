# Independent Research Module

独立研究闭环模块，面向研究问题分析、迭代搜索、AI 提炼和网页阅读展示。

## 当前结构

- `backend/`：FastAPI 后端、配置中心、搜索与报告生成逻辑
- `frontend/`：React + Vite 前端工作台与报告阅读器
- `docker/`：Docker Compose 样例
- `docs/`：使用说明和部署说明

## 能力概览

- 结构化问题分析
- 多轮并行搜索与覆盖度评估
- 外部 AI 兼容调用与本地回退
- 长文报告阅读器与设置页配置
